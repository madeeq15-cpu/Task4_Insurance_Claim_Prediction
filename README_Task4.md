# Task 4: Predicting Insurance Claim Amounts

## Project Overview
This project predicts medical insurance charges using a Linear Regression model based on personal attributes such as age, BMI, smoking status, and region.

## Dataset
- Dataset: insurance.csv
- Target Variable: charges

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook / VS Code

## Methodology
1. Data loading and inspection
2. Exploratory Data Analysis (EDA)
3. Encoding categorical variables
4. Training a Linear Regression model
5. Model evaluation using MAE, RMSE, and R² score

## Key Insights
Smoking status, age, and BMI significantly influence insurance charges.

## Conclusion
This project demonstrates regression modeling and healthcare cost prediction using real-world data.
